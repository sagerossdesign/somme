### How to use skills
- Trigger rules:
  - Use `brand-site-generator` for direct brand site generation in current project.
  - Use `site-foundation-architect` when user asks to start a new project type from foundation/template.
- Do not carry skills across turns unless re-mentioned or clearly still in the same request.
- If required details are missing, make reasonable assumptions and continue with placeholders.
- Always test locally first before recommending deploy.
