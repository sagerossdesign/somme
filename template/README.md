# Codex Website Template Kit

This folder contains reusable templates so you can spin up new brand websites with Codex quickly.

Files:
- `BRAND_INPUT_TEMPLATE.md`: fill-in brief for each new brand
- `CODEX_PROMPT_TEMPLATE.md`: reusable instruction prompt for Codex
- `WORKFLOW_CHECKLIST.md`: repeatable local-first build/deploy flow
- `ONE_SHOT_THREAD_PROMPT.md`: copy/paste prompt for new threads

Recommended process:
1. Fill brand input template.
2. Paste prompt template into Codex + include your filled input.
3. Implement and test locally.
4. Commit and deploy.
